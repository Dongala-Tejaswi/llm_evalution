# GroqEval - LLM Evaluation Dashboard

An automated LLM evaluation platform built using:

- Groq API
- Streamlit
- BLEU Score
- Semantic Similarity
- ROUGE
- Latency Metrics

## Features

- User enters a question
- User enters a reference answer
- Groq generates an answer
- Automatic evaluation
- Interactive dashboard

## Tech Stack

- Python
- Groq API
- Streamlit
- NLTK
- Sentence Transformers

## Run

pip install -r requirements.txt

streamlit run src/app.py
